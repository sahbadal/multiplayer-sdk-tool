# Out of Scope

The Unity Multiplayer SDK focuses on networking infrastructure and developer tooling. The following systems are intentionally outside the scope of the SDK:

## Gameplay
- Character movement implementation
- Combat systems
- Weapons
- Damage systems
- Health systems
- AI / NPC behaviour
- Game rules
- Game progression
- Quests and missions

---

## User Interface
- Main Menu
- Lobby UI
- Matchmaking UI
- Room Browser UI
- Voice UI
- Chat UI
- Emoji UI
- HUD
- Inventory UI

---

## Animation
- Character animations
- Animator Controllers
- Blend Trees
- IK systems

---

## Backend
- Authentication
- Player accounts
- Databases
- Cloud Save
- Economy
- Leaderboards
- Inventory services
- Analytics
- Dedicated Servers

---

## Networking Logic
The SDK will NOT automatically implement:
- Character controllers
- Vehicle controllers
- VR locomotion
- Custom prediction algorithms
- Lag compensation
- Rollback networking
- Game-specific RPC logic
- Custom synchronization algorithms

---

## Art & Content
- Models
- Materials
- Audio assets
- Prefabs
- Scenes
- Particle effects

---

## What the SDK DOES Provide

- Session Management
- Matchmaking APIs
- NetworkRunner Management
- Photon Voice Integration
- RPC Helpers
- Network Event System
- Player Spawn Workflow
- Prefab Validation
- Scene Validation
- Project Validation
- Auto-Fix System
- Diagnostics
- Centralized Configuration
- Editor Tooling
- Standard Multiplayer Architecture
